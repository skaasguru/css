CSS Inheritance Example

Inheritable Property Example (color)

This text inherits the color property from the parent container.

All text inside this div will inherit the same color, making it blue.

Non-inheritable Property Example (margin)

The margin property is not inherited. This paragraph will not have the same margin as the parent container.

Using the `inherit` Value

This paragraph inherits the background-color from its parent element.

Using the `initial` Value

This paragraph uses the `initial` value for the color property, which resets it to the default browser value (black).

Using the `unset` Value

This paragraph uses the `unset` value for font-size. It either inherits the font size or sets it to its initial value, depending on context.

Text Property Inheritance (font-size, line-height, font-weight)

This paragraph inherits the font-size, line-height, and font-weight from the parent element.

The text is bold and properly spaced due to inheritance of these properties.

Non-Inheritable Properties

The border and padding properties of this element are not inherited by child elements. You will need to apply them explicitly.
